# Roadmap de desarrollo

## Fase 1: Base del proyecto

- Crear repositorio.
- Crear backend Spring Boot.
- Crear frontend Angular.
- Configurar PostgreSQL.
- Agregar script inicial de base de datos.
- Configurar variables de entorno.
- Configurar login JWT.

## Fase 2: Seguridad y usuarios

- Login.
- Roles.
- Usuarios.
- Guards frontend.
- Interceptor JWT.
- Redirección inicial a login.

## Fase 3: Catálogos principales

- Productos.
- Categorías.
- Unidades de medida.
- Clientes.
- Proveedores.

## Fase 4: Inventario

- Stock actual.
- Movimientos de inventario.
- Ajustes manuales.
- Alertas de stock bajo.

## Fase 5: Ventas / POS

- Escaneo de código de barras.
- Carrito de venta.
- Cantidad editable.
- Métodos de pago.
- Descuento automático de inventario.
- Registro de ingreso en caja.

## Fase 6: Compras

- Registro de compras.
- Detalle de compra.
- Aumento de inventario.
- Registro de egreso en caja si aplica.

## Fase 7: Caja y Finanzas

- Apertura de caja.
- Cierre de caja.
- Ingresos.
- Egresos.
- Diferencias.
- Movimientos manuales.

## Fase 8: Reportes

- Ventas por día.
- Ventas por método de pago.
- Productos más vendidos.
- Stock bajo.
- Ingresos y egresos.
- Utilidad estimada.

## Fase 9: Mejoras futuras

- Carrito virtual público.
- Página pública de productos.
- Facturación.
- Impresión de tickets.
- Multi-sucursal.
- Backup automático.
- Dashboard gerencial.
