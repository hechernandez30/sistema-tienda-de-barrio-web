# Capturas del Manual de Usuario

Esta carpeta contiene las capturas de pantalla utilizadas en `manual-usuario.md`.

Coloca aquí las imágenes con los nombres exactos indicados en
[`../capturas-pendientes.md`](../capturas-pendientes.md), por ejemplo:

- `01-login.png`
- `02-dashboard.png`
- `03-productos-listado.png`
- ...

Formato recomendado: **PNG**, ancho aproximado de **1280–1600 px**.
